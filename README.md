# 🏗 팀 소개 웹페이지

이 프로젝트는 **팀을 소개하는 웹페이지**로, HTML, CSS, Bootstrap을 활용하여 제작되었습니다.  

---

## 📌 주요 기능

✅ **팀 소개 페이지**  
- Bootstrap을 활용한 반응형 디자인  
- 팀원별 프로필 카드 스타일 구성  

✅ **개인 소개 페이지**  
- HTML & CSS를 활용한 간단한 자기소개  
- 깔끔한 레이아웃 적용  

✅ **카카오 지도 API 활용**  
- 현재 위치를 가져와 지도에 마커 표시  
- 줌 컨트롤 & 이동 기능 지원  

---

## 🛠 사용된 기술

| 기술          | 설명                                      |
|--------------|------------------------------------------|
| `HTML`      | 웹페이지 기본 구조 생성                 |
| `CSS`       | 스타일 및 레이아웃 디자인 적용          |
| `Bootstrap` | 반응형 웹 디자인 적용                   |
| `Kakao Maps API` | 지도 기능 및 현재 위치 마커 표시 |
